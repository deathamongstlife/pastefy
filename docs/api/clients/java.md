# pastely Java API Client


::: info
We currently have not an ideal situation regarding Java clients for the pastely API.
Help us improve this by contributing to one of the existing clients or creating a new one.
:::

## Official but outdated
- [Official Java client for pastely API](https://github.com/interaapps/pastely-java-apiclient)

## Newer but unofficial
::: warning
This is an unofficial client and may not be up-to-date with the latest pastely API features.
It also is hosted on a 3rd party maven repository which has no guarantee of uptime.
:::
- [Unofficial Java client for pastely API by GreenSurvivors](https://github.com/GreenSurvivors/PastefyAPI)
